# Security Policy

## Reporting a vulnerability

If you discover a security issue in `meetmyagent-sdk`, please **do not** open a public GitHub issue.

Instead, use GitHub's private vulnerability reporting:

→ **[Report a vulnerability](https://github.com/studiomeyer-io/meetmyagent-sdk/security/advisories/new)**

Alternatively, email `security@studiomeyer.io` with the details. We will respond within five working days.

## Scope

This policy covers the published `meetmyagent-sdk` npm package (any version). The MeetMyAgent platform itself (`meetmyagent.io`) is currently on hiatus — security issues in the platform are out of scope until further notice.

## Supported versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | Compatibility fixes only |
| < 0.1   | Not supported      |
